# Todo List

- [ ] Collect Inbox Tasks
  - Gather all outstanding tasks, notes, and ideas from email, notes, and brain-dump into a single list.
- [ ] Categorize Tasks
  - Group tasks into Work, Personal, Errands, Learning, and Backlog to clarify context.
- [ ] Prioritize (Eisenhower)
  - Label each task as Urgent/Important, Important/Not Urgent, Urgent/Not Important, or Neither.
- [ ] Estimate & Schedule
  - Assign time estimates and add top-priority tasks to calendar/time blocks this week.
- [ ] Break Down Large Tasks
  - Split projects into 15–60 minute actionable subtasks with clear next actions.
- [ ] Set Deadlines & Reminders
  - Add due dates, reminders, and notifications for time-sensitive items.
- [ ] Weekly Review
  - Every week review completed items, update priorities, and plan the next week.
